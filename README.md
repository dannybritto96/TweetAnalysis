# Tweet Analysis Django App

- Uses NLTK libraries for Sentiment Analysis
- Sentiment Analysis on last 200 tweets
- Sentiment Analysis on last 200 mentions
- Most used words
- Most used hashtags
- Tweet numbers by languages
